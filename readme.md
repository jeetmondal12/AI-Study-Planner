# AI Study Planner

A full-featured AI Study Planner web app built as a single React JSX file using Tailwind CSS and the Anthropic API.

## Features

### Core AI Features (Powered by Anthropic Claude)
- **AI Study Plan Generator**: Generates structured day-by-day study plans based on topic.
- **AI Flashcard Creator**: Extracts key concepts from text to create Q&A flashcards.
- **AI Quiz Generator**: Creates multiple-choice quizzes on any topic.
- **AI Study Buddy Chat**: Conversational AI tutor for Q&A.
- **AI Summary Generator**: Summarizes long notes into bullet points.
- **AI Weak Spot Detector**: Provides personalized feedback based on quiz performance.

### Productivity Tools
- **Subject Manager**: Organize study materials by subject with color coding.
- **Task & To-Do List**: Track assignments and deadlines.
- **Pomodoro Timer**: Focus timer with break intervals.
- **Study Streak Tracker**: Visual streak counter.
- **Progress Dashboard**: Charts and analytics for study hours and quiz scores.
- **Notes Section**: Rich text-like notes area per subject.
- **Calendar View**: Monthly calendar integration.
- **Flashcard Deck Viewer**: Interactive flip cards.
- **Dark/Light Mode**: Theme toggle.

## Tech Stack
- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Charts**: Recharts
- **Animations**: Framer Motion
- **Date Handling**: date-fns
- **AI**: Anthropic API (Claude Sonnet)

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start the development server.
4. Run `npm run build` to build for production.

## Note
This application is designed as a single-file React component (`src/App.tsx`) for portability, while maintaining a modular internal structure.
